<h2 data-pm-slice="0 0 []">一、2026 年二季度货币政策委员会例会核心要点总结</h2>
<p data-pm-slice="0 0 []"><strong>（一）经济形势判断</strong></p>
<p data-pm-slice="0 0 []"><strong>1.外部环境</strong>：对国际形势复杂性和不确定性的关注上升。外部环境更加复杂多变，全球增长动能偏弱，地缘、经贸冲突频发；主要经济体经济表现分化、通胀、货币政策显著分化，海外政策调整不确定性上升。</p>
<p data-pm-slice="0 0 []"><strong>2.国内经济：</strong>注意到国内 K 型增长，科技板块向新向优，发展成效显著，但也关注到当前的结构分化。</p>
<p data-pm-slice="0 0 []"><strong>（二）央行基调</strong></p>
<p data-pm-slice="0 0 []"><strong>1.总基调：</strong>继续实施适度宽松货币政策，加大逆周期、跨周期调节，总量 &nbsp; 与结构并重。<strong>总基调决定了央行流动性收紧只是暂时的，未来进一步收流动性可能性不大。结合5、6 月份调控思路，核心还是以资金调整“时间”换利率调控的“空间”。</strong></p>
<p data-pm-slice="0 0 []"><strong>2.政策出台：</strong>强调增量+存量政策集成效应，增强政策前瞻性灵活性针对性。<strong>未来央行一方面配合财政发债，创造良好的流动性环境；另一方面可能主动发力，进一步优化当前的利率调控，规范贷款行为，降低融资中间费用，核心是提升存量资金使用效率，从而降低企业融资成本。存量资金已足够充裕，降准、降息仍是应对性政策，依靠结构性工具发挥作用。</strong></p>
<p data-pm-slice="0 0 []"><strong>3.宏观审慎：</strong>特意强调从宏观审慎视角持续评估债市运行，重点跟踪长期国债收益率波动，防范债市扰动流动性传导。<strong>金融稳定是大事情，未来人民银行利率调控的范围和能力都会逐步提升，收益率难以快上快下，交易机会可能相应减少，配置更具性价比。</strong></p>
<p data-pm-slice="0 0 []"><strong>4.对外开放：</strong>汇率、开放均以稳为主，开放前提是安全、高水平，意味着围绕<strong>开放可能进一步要提高金融管理能力、风险防控能力。</strong></p>
<h2 data-pm-slice="0 0 []">二、二季度 vs 一季度例会核心差异对比</h2>
<p data-pm-slice="0 0 []"><strong>1.经济判断</strong>：一季度未突出结构分化；二季度将 “结构分化” 列为国内核心矛盾，意味着结构性信贷、定向工具权重提升。</p>
<p data-pm-slice="0 0 []"><strong>2.操作思路：</strong>一季度侧重工具投放；二季度强调预判、动态微调、相机抉择，从“被动对冲”转向“主动前瞻调节”，政策操作弹性提升。</p>
<p data-pm-slice="0 0 []"><strong>3.债市管理：</strong>一季度仅笼统提及金融市场；二季度单独明确：宏观审慎视角监测债市、重点关注长期收益率，债券市场波动纳入常态化观测指标。</p>
