<p class="t-content__chapo">
            <span class="t-location">美国 – </span>            美国国会两名跨党派众议员提出《云端安全法》（Cloud Security Act），拟填补现行人工智能（AI）芯片出口管制漏洞，防止中国等美国对手透过租用美国云端服务取得先进AI运算能力，绕过高阶芯片出口限制。        </p>
<p class="m-pub-dates"><span class="m-pub-dates__date">发表时间： <time datetime="2026-06-27T18:23:28+00:00" pubdate="pubdate">27/06/2026 - 20:23</time></span></p>
<p>法案由民主党籍新泽西州众议员乔希．戈特海默（Josh Gottheimer）及共和党籍密西根州众议员、众议院中国问题特别委员会主席约翰．穆勒纳尔（John Moolenaar）共同提出。</p>
<p>两人指出，美国目前限制向特定国家及实体，出口用于训练先进人工智能模型的高阶AI芯片，但客户仍可透过美国云端平台租用搭载这些芯片的运算资源，形成出口管制漏洞。</p>
<p>根据法案内容，美国云端服务供应商若怀疑来自美国对手国家的客户，利用其平台发展先进人工智能模型，可自愿向商务部通报相关情况，协助政府防堵规避出口管制的行为。</p>
<p>法案并将修订《储存通讯法》（Stored Communications Act），解决现行法律限制云端业者，主动向政府揭露客户资料所造成的法律疑虑，让企业在发现可疑情况时可依法通报。</p>
<p>戈特海默表示：“我们不能让对手，尤其是中国，只因为租用而不是购买，就能规避美国出口管制。”</p>
<p>他说：“这项法案让美国企业拥有明确的法律依据，在发现不法行为者企图利用我们自己的云端基础设施威胁美国国家安全时，可以依法向政府通报，做正确的事。”</p>
<p>穆勒纳尔表示，中国正积极寻找替代途径取得先进运算能力。</p>
<p>他说：“在AI竞赛中，中国会购买能买到的一切，其余的就会偷走，因此正积极试图透过后门方式存取美国资料中心，并利用云端运算训练AI模型。”</p>
<p>他并表示：“美国云端平台有责任阻止中国AI能力持续扩张，因为这将助长其军事及监控野心。这项跨党派、符合常识的法案，只要求业者确认使用者身分，就能同时保护自家产品与美国国家安全。”</p>
<p>法案主要适用于提供高效能AI运算服务的美国云端平台，包括亚马逊Amazon Web Services（AWS）、微软Azure及Google Cloud等业者。提案议员认为，若仅限制实体芯片出口，未同步管理云端算力租用，出口管制仍存在重大缺口。</p>
<p>这项法案提出之际，特朗普（Donald Trump）政府正研拟人工智能政策，美国国会近年也持续推动AI相关立法，除加强出口管制外，也聚焦深度伪造（Deepfake）、AI仿冒内容、儿童使用AI安全及人工智能对经济的影响等议题。</p>
<p>事实上，共和党籍联邦参议员戴夫．麦考密克（Dave McCormick）及民主党籍联邦参议员荣恩．怀登（Ron Wyden）2025年也曾提出《远端存取安全法》（Remote Access Security Act），同样希望限制外国势力透过美国云端服务取得先进AI芯片运算能力。</p>
