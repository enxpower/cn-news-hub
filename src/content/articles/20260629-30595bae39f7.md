<h2 class="post__body__extend__item__title" data-v-7653d8d5="">Marked 3：Markdown 高级预览工具</h2>
<ul><li>平台：macOS</li><li>关键词：Markdown、预览器</li></ul>
<p>@ElijahLee：如果你经常使用 Obsidian、VS Code、Bear、Ulysses 或其他 Markdown 编辑器，Marked 3 可以说是一款非常独特的辅助工具。 它不是 Markdown 编辑器，而是一个专注于实时预览、校对、排版和导出的伴侣应用。</p>
<p>很多 Markdown 软件都是「编辑器 + 预览器」一体化，例如 Typora、iA Writer，在编辑时所见即所得。Marked 3 采取完全不同的思路，它允许在你熟悉的软件中写作，对 .md 文件进行编辑，而此时 Marked 3 会通过一个独立的窗口， 预览排版效果。并且它会实时监听 .md 文件的变化，一旦在编辑器中进行保存，Marked 3 实时刷新预览效果。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/0df6cdc0fe028517f89a22e90429dca9.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>因此，它可以配合几乎所有编辑器使用，包括：Obsidian、VS Code、Scrivener、Ulysses 等等，只要能够保存 Markdown 文件的软件，都可以作为它的前端编辑器。</p>
<p>Marked 3 对 Markdown 的支持非常完整，几乎覆盖目前主流 Markdown 语法生态，包括CommonMark、GitHub Flavored Markdown（GFM）、MultiMarkdown、Kramdown 等等。它还能支持正确显示 Mermaid 流程图、MathJax、KaTeX 数学公式、CriticMarkup 等等。正如应用的宣传语，Marked 3 可以说是终极 Markdown 文件的预览工具。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/1b90d5d93658bc2f8ad913b0b724d194.JPEG?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>高度的自定义能力，使得我们可以按照自己的需求重新定义 Marked 3 的整个渲染流程。应用的 Style Manager 提供 9种不同样式风格，自定义 Processor 可以选择 Markdown 语言和 HTML 格式，自定义转换规则（Custom Rules）可以定制样式、调整元数据等。这些功能对于高度专业的 Markdown 用户、博客作者、电子书作者等等非常有用。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/7001f1d21b121566e57c5864f43e96a1.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>强大的导出能力也是 Marked 3 的优点，它支持将 .md 文件导出为 PDF、HTML、DOCX、EPUB、RTF、TextBundle 等 9 种格式。第三代最大的升级之一，就是重新设计了 DOCX 导出和导入能力，支持保留批注、修订、数学公式、图片等。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/334328ab4ca25170b1e666fcf64e70b2.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>Marked 3 并不是为了取代 Obsidian、Typora 或 VS Code，而是定位于它们之上的专业预览和出版工具。它最大的优势在于兼容几乎所有 Markdown 编辑器，并提供更完整的排版预览、校对分析、导出能力以及高度可定制的工作流。对于经常创作长文、技术文档或需要输出高质量 PDF、Word、EPUB 的用户来说，它能够显著提升 Markdown 写作体验。</p>
<p>你可以在 App Store 免费下载 Marked 3，应用提供 7 天免费试用，结束后必须付费订阅才可继续使用。费用为 22 元 / 月、198 元 / 年，498 元买断。Marked 3 还收录在 Setapp 中，Setapp 用户可以免费下载使用。</p>
<h2 class="post__body__extend__item__title" data-v-7653d8d5="">Juicy 1.5 ：更全面的 Mac 电源管理应用</h2>
<ul><li>平台：macOS</li><li>关键词：电源管理、充电优化</li></ul>
<p>@化学心情下2：相较 Windows 笔记本在电源管理上的能力，macOS 在电池策略层面已经提供了较完整的原生支持。例如在「设置 - 电池」中，用户可以查看电池健康状态，并启用「优化电池充电」，系统会在日常使用场景下自动延缓充电至 80% 以上。同时也支持充电上限设置，用于控制长期充电行为。对多数用户而言，这些能力已经足够覆盖日常需求。</p>
<p>但如果需要更细的电池健康分析，例如循环次数变化、应用级耗电详情，或者更精确的电池状态监测，则通常需要借助第三方工具，定位为 macOS 电池监控与分析的 Juicy 便是其中之一。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/3285468097d4cbbdcc4cae438edf3a19.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>在最新的 v1.5 版本中，Juicy 强化了充电控制能力，新增 50% 到 80% 的充电上限区间设置。对于长期接入电源的 MacBook 使用场景，这类限制可以在一定程度上降低电池循环累积，相比系统默认的 80% 优化策略，可以进一步将上限压低至 60% 等更保守的区间，从而延缓电池损耗。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/635b68d8d59019ee2b546ddd404edffe.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>在电池使用分析方面，Juicy 延续并扩展了 macOS 的基础能力。通过 App Usage 模块，可以查看应用在 24 小时、7 天及 30 天维度的耗电表现。进一步点击单个应用后，还能看到其耗电占比、峰值消耗以及功耗变化曲线，用于定位高耗电来源。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/730cb524b37c2b0cb02a073262fd5801.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>除了 Mac 本机数据，v1.5 还增加了对 iPhone 与 iPad 电池信息的读取能力。当设备通过 Wi-Fi 或 USB 与 Mac 连接后，Juicy 可以展示包括电池循环次数、当前电量、电池健康度、电池温度，以及设计容量与实际容量等信息，同时还可显示电池生产信息与序列号等更底层的数据。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/534dc314c5634b7074439dfe0dd70451.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>本次更新还加入了 Power Flow 模块，用于展示充电功率的实时变化，以及电流从适配器进入系统与电池分配的路径。同时，电池图标样式也支持自定义，包括颜色、横竖形态，以及是否显示电量百分比等显示选项。</p>

<h2 class="post__body__extend__item__title" data-v-7653d8d5="">iA Writer 8：不止是外观更新</h2>
<ul><li>平台：iOS / iPadOS / macOS / Windows</li><li>关键词：笔记、写作</li></ul>
<p>@Peggy_：现如今大家的目光已经被双链笔记、第二大脑等概念彻底吸引，早年间以编辑体验优秀闻名的 Bear、iA Writer 等软件似乎也逐渐被遗忘。就在本月，写作软件 iA Writer 迎来了 8.0 的大版本更新。</p>
<p>在各式软件堆叠功能、拥抱 AI 的当下，iA Writer 仍然像是一个久居深山的老者，默默耕耘自己的一方天地，此次更新便也是围绕和写作相关的主要功能展开。首先是外观更新，iA Writer 紧跟平台设计风格，先后花费一年时间逐步将 Liquid Glass 风格融入软件设计当中。这一点在 iOS 平台尤为明显，原本在编辑页面底部以长条状显示的页面工具栏现在被拆分为多个单独的按钮，变为了独立圆形玻璃胶囊按钮，相比于旧版本，新版设计与系统的融合程度更佳。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/1baf589ae237944c7b25d7e5c9863653.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>此次最为主要的功能更新便是将搜索与文档大纲的融合，在编辑界面点击搜索按键，弹出菜单里会显示当前文件的名称及文档大纲。更为方便的是，搜索页面不但可以检索我们以往的所有文档，还能精确搜索出某一文档下的具体标题位置，方便我们在写作时随时检索以往内容。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/e3adb015fdbab313ec644efef97cd47e.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>在之前的应用介绍文章当中我着重介绍了 iA Writer 的命令键盘功能，然而在更新至 8.0 版本后，原本是「命令键盘」的位置变更为了「命令面板」，熟悉的快捷操作不见了，取而代之的是菜单样式的功能选项，许多原本要去应用设置内寻找的选项都被整合到了命令面板当中，当然，命令面板仍然支持自定义，我们可以根据自身的偏好将 iA Writer 打造为自己的专属写作工具。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/3ff717c1cb34049af8ba5dd5ead099d0.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>如果你不习惯这一变化，可以去设置当中将命令按键的操作变更回「命令键盘」，但 iA Writer 也提到会在后续的更新当中移除命令键盘功能，因此我们不妨早日适应新的界面，从头打造一个趁手的界面。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/6bd7db4589a9aa15791935ff72d1a0b1.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>你可以通过 App Store 下载体验 新版 iA Writer，或在官网获取 Windows 版本。</p>

<h2 class="post__body__extend__item__title" data-v-7653d8d5="">Soulver 4：这一次，它不只是帮你计算</h2>
<ul><li>平台：iOS / iPadOS / macOS</li><li>关键词：计算器</li></ul>
<p>@Snow：Soulver 绝对算得上 Apple 应用生态甚至是 AI 算法中的「老人」了，早在十几年前，它就能「看懂」混杂着各种表述和思考的「人话」，快速计算出你想要的结果，仿佛计算草稿成了精。而如今，我们已经习惯了各种 AI 精帮我们整理信息、生成答案，甚至代替我们完成部分思考时，Soulver 时隔 7 年带来的 4.0 大版本更新，似乎并没有试图成为另一个给你答案的 AI 助手，而是试图拓宽这张「计算草稿纸」，让它承载更多问题的思考过程。</p>
<p>多年前更新的 3.0 版已然支持日期计算，4.0 版在此基础上增加了时间点标记，使得 Soulver 也可以承担起简单的日程规划任务。以下图为例，我的起床闹钟设为 8:30，航班时间点定为 14:30，我就可以以这两个时间为锚点，结合我规划的行程，让 Soulver 帮忙判断来不来得及。我可以随时在这张草稿上划划改改，随时调整时间节点前的任意行程与耗时，它都会帮我重新计算每项活动的时间安排，以及是否出现超时的隐患。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/dc172fbbbe35ccfcedabcd57fbcc514d.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>你也可以将日程安排，与消费规划相结合，调整行程日期的同时，同步调整消费预算，无需等待 AI 助手重新跑流程，快速就能看到预算的变化。</p>
<p>标签工作流也是 4.0 版更新重点之一，它允许你通过为某些行加上标签，实现快捷的统计操作。比如你与好友一同出游时，可以通过标签来区分二人各自的消费支出，在旅程结束时快速统计，明确各自需要 AA 的份额。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/5557d31b455d6b2de01db9acfb010685.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>另一项可以体现 4.0 草稿拓宽的更新，就是加入了自定义全局表单，你可以在这里定义全局适用的变量和新的单位类型。</p>
<p>此前使用 Soulver 进行计算时，往往习惯于将需要用到的所有变量置顶写好，以便后续算法调用，但如果你有多张草稿都需要调用这一数据，则需要在每个张草稿前面都贴一次。更新之后，你可以将这些常用的变量添加在全局表单中，所有草稿都可以按需调用这一变量，无需再做「事先声明」。如果数据有变动，你也只需要在全局表单中修改即可，所有算法都会在修改后自动更新。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/b7da96a11c9ba514f569fbbcefc33d22.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>此外，你还可以在这里设置新的等价关系和单位类型。比如小约翰可汗常用单位，你就可以在这里将其与我们常用的平方千米设置为等价关系，之后就可以快速计算出每个区域大概等于多少个通辽了。而一些无法通过等价关系计算的新单位，例如 token，你可以在全局表单中使用 new unit 作为其值，创建一种全新的单位类型，方便你开展更适配你自身需求的运算。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/8d4d85e32c64f490c4b4478260b178ae.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>在编辑体验上，4.0 版增加 Markdown 编辑器会高频使用的高亮突出和分割线功能。这与 Soulver 适配度极高，既然它记录的是一整段不断演化的思考过程，当这张可以无限延长的稿纸里同时存在记录、变量、假设、计算时，高亮可以帮助你快速抓住重点变量、算法和结果，而分割线则让不同区块边界变得清晰，尽可能确保这张稿纸不会失控成一堆数字堆叠。</p>

<h2 class="post__body__extend__item__title" data-v-7653d8d5="">Grove：用好习惯养成一颗「树」</h2>
<ul><li>平台：Android</li><li>关键词：习惯养成</li></ul>
<p>@大大大K：以前用过不少习惯养成类型的 App，不过它们大多数都是用打卡按钮的形式，每天点一下确实很无聊，甚至时间久了，打卡动作反而变成了一种负担。之前也尝试过 Habitica 这种游戏类型的习惯养成类 App，但对我来说稍显复杂。而我最近发现的 Grove 这款 App 则恰好处于我的甜点区，它用「生长树」的形式将打卡成就可视化，但又不需要非常复杂的养成。</p>
<p>正如其名字所示，Grove 摒弃了传统打卡 App 那种枯燥的数字和对勾显示，而是将你的习惯养成具象成一棵基于分形几何算法的、不断生长的「习惯树」。在 Grove 里，每次你「种一颗新树」，都可以选择两种不同的生长机制：如果你要养成好习惯（比如每日健身、早睡早起），那就选择「打卡模式」，每完成一次任务就会让小树长高、分叉，最终长成一棵茂盛的大树，树上的每一个枝叶都是你努力的证明。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/52fc8b58d54b70eb15ba65b682ff6a26.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>如果是为了戒掉某些不良习惯（比如戒酒、戒烟），那就用「戒断模式」。在该模式下，你只需要坚持不做这件事，小树会自己生长，一旦你忍不住「破戒」了，那就要去 Grove 里记录「复发」的时间和原因，然后亲眼看着自己养成的树苗枯萎成一颗种子。这种眼睁睁看着自己亲手培育的植物瞬间凋零的挫败感，我觉得要比单纯的警告更有效。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/769d2a1752891303be6f9b859803ed19.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>既然每天都要看着这棵小树一天天成长，界面的颜值和操作体验就显得尤为重要。Grove 在这方面算是比较精致了。它适配了 Material You 并且支持动态取色功能，支持自定义习惯卡片的显示方式，其中「滚轮」显示模式比较有趣，它带有 3D 效果，上下滚动时配合线性马达清脆的震动，每次操作都非常解压。进入卡片后可以看到小树具体的成长情况、打卡历史以及日历展示，也可以在这里使用「请假」功能（比如某几天出差或者生病，就可以在健身卡片中请假）。</p>
<img src="https://cdnfile.sspai.com/2026/06/29/article/98beb362a5381d0420b62b0084eb9f7f.jpeg?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>整体而言，Grove 用这种培育小树来追踪习惯的方式，既免去了传统打卡应用的无聊，又不像大型习惯养成类 App 那样繁琐，每天看着它一点点发芽生枝，能让人感受到一种富有生命力的成就感。如果你也想要找一个简洁纯粹又不枯燥的习惯养成工具，可以在 GitHub 和 F-Droid 免费下载 Grove。</p>
