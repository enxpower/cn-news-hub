<p>距离 Claude Fable 5 被美国政府一纸出口管制指令强行下线，已经过去了将近两周。</p>
<p>这两周里，行业里有一句流传颇广的玩笑话：现在用 AI 最高效的方式，就是什么都不做，坐在沙滩上等 Fable 回来。</p>
<p>但这两天，等待似乎有了一点实质性进展。</p>
<h2>Claude Code 悄悄改了几个字符串</h2>
<p>6 月 24 日，有人在 Claude Code v2.1.190 的二进制文件里发现了一些不寻常的字符串变更。新增的提示文案是「您本周的 Fable 5 使用额度已用完，继续使用 Fable 5 将消耗使用积分」），而原本的「需单独购买，不包含在当前套餐内」字样被悄悄移除。</p>
<img src="https://wpimg-wscn.awtmt.com/aa863558-f886-45ac-b475-d1a0d89656f1.png?imageView2/2/w/680" alt="" loading="lazy">
<p>这个发现似乎最早由 X 用户 @synthwavedd 公开，其推文浏览量已经突破百万，表明大家都非常关心这件事。</p>
<img src="https://wpimg-wscn.awtmt.com/b2eb3a0a-84af-4025-af59-110fa0c8867a.png?imageView2/2/w/680" alt="" loading="lazy">
<p>科技媒体 Decrypt 随后用 npm pack 下载了 Claude Code 的 macOS Apple Silicon 包，解压后对内置的 claude 可执行文件运行 strings -a 命令，确认了这些字符串确实存在于生产二进制文件中，却并不在 GitHub 公开的更新日志里。</p>
<img src="https://wpimg-wscn.awtmt.com/eaa72a30-de76-4157-b55e-64449a621f1b.png?imageView2/2/w/680" alt="" loading="lazy">
<p>这类字符串变更本身说明不了太多，因为产品团队提前几个月写好功能文案是常规操作，代码变化不代表明天就会上线。但它至少传递了一个明确信号：Fable 5 不会像最初设想的那样，作为订阅计划之外的单独付费项目存在，而是会被重新整合进 Pro、Max、Team 等订阅套餐的常规额度内，按周重置。</p>
<p>这与 Fable 5 最初的商业化路径（两周免费体验后转为按使用积分单独计费）形成了明显反差。</p>
<p>更有意思的是，已经有用户注意到，Amazon Bedrock 的模型目录页面上，Fable 5 的条目仍然挂在那里，标注为「由 Anthropic 出售」。</p>
<img src="https://wpimg-wscn.awtmt.com/f9e262ef-2335-486f-ac9e-3becf180c08c.png?imageView2/2/w/680" alt="" loading="lazy">
<img src="https://wpimg-wscn.awtmt.com/cbb12215-90fb-4a1b-bc9b-33cddbb19f53.png?imageView2/2/w/680" alt="" loading="lazy">
<p>结合 Claude Code 里的字符串变化，不少观察者认为，Fable 5 可能真要回归了。</p>
<img src="https://wpimg-wscn.awtmt.com/ffd4c8cc-c4fd-47e4-89bb-624e17692d7b.png?imageView2/2/w/680" alt="" loading="lazy">
<p>事实上，到了今天下午 3 点多时，已经有少量用户在X上表示 Fable 5 已经可用了。但我们目前还没看到更可靠信源的确切消息。</p>
<img src="https://wpimg-wscn.awtmt.com/d4d94233-06a6-40c2-b50b-17090b689619.png?imageView2/2/w/680" alt="" loading="lazy">
<h2>「不要再让 Amodei 去谈了」</h2>
<p>伴随 Fable 5 可能回归的消息，白宫那边却传来了一个颇有戏剧性的人员替换。</p>
<p>据《连线》报道，美国政府近来与 Anthropic 的沟通明显顺畅了不少，原因很简单：他们不必再跟 CEO Dario Amodei 打交道了。<strong>在所有关于 Fable 5 重新上线的高层会议中，Amodei 已经被他的联合创始人 Tom Brown 取代。</strong></p>
<p>一位直接参与相关通话的人士说：「Tom Brown 不会像 Dario 那样表现得像个『怪人（weirdo）』，他真的能正常沟通。」</p>
<img src="https://wpimg-wscn.awtmt.com/3b96a843-47ec-4749-bf30-6ab6bcd4fe01.png?imageView2/2/w/680" alt="" loading="lazy">
<p>Brown 的正式职务是 Anthropic 联合创始人兼首席计算官（Chief Compute Officer），自 6 月 15 日起，他与公司公共政策负责人 Sarah Heck 一起常驻华盛顿，与商务部长卢特尼克、国家网络总监 Sean Cairncross 等官员展开工作组级别的谈判。</p>
<p>Anthropic 红队负责人 Logan Graham 与资深安全研究员 Nicholas Carlini 也亲赴华盛顿，带去了具体的技术团队，而不只是公关层面的姿态。</p>
<p>转折点似乎可以追溯到 Brown 接手谈判之后。6 月 19 日，特朗普在接受 Axios 采访时改口表示，他不再认为 Anthropic 或 Amodei 本人是国家安全威胁；而就在一周前，他的措辞还要严厉得多。</p>
<p>「现在不是了，但一周前可能算。」特朗普这样说。</p>
<h2>结语</h2>
<p>当然，目前所有公开信息都还停留在「信号层面」。</p>
<p>Claude Code 的字符串变化、Bedrock 目录中的保留条目、以及华盛顿谈判团队结构调整，这些信息彼此独立，但方向高度一致：Fable 5 并未真正消失，而是在重新定义其商业与政策位置。</p>
<p>如果把这些碎片拼起来，一个更现实的可能性是：Fable 5 不会「恢复原状」，而是以一种更深度嵌入订阅体系的方式重新上线，同时还在监管框架下调整访问与计费结构。</p>
<p>对此，你怎么看？</p>
