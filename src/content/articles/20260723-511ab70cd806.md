<p>
                                                在当今世界和平被战争所破坏、国际地缘政治局势错综复杂、诸多国家在政治与经济层面遭遇重大挑战的背景下，正在菲律宾首都马尼拉召开的东盟外长会议罕见成为了中美俄大国展开地缘政治角逐的外交舞台，其中关于结束俄乌战争的美俄会谈尤其引人瞩目。                    </p>
<p>东盟外长会议-“携手共创未来”</p>
<p>第59届东盟外长会议（AMM-59）以及系列会议于2026年7月20-24日在马尼拉的菲律宾国际会议中心（PICC）举行。这是东南亚国家联盟（ASEAN）重点讨论区域一体化、地区安全挑战以及对外关系的重要平台。本次会议由东盟轮值主席国菲律宾主持，主题为“携手共创未来”。</p>
<p>时值《东南亚友好合作条约》签署50周年，东盟持续扩员，东帝汶作为第11个新成员国正式签约。同时还接纳土耳其为东盟第12个对话伙伴。东盟对话伙伴国还包括中国、日本、韩国 、美国 、俄罗斯、印度、澳大利亚、新西兰、加拿大、英国、欧盟。此外，联合国 (UN) 也是东盟重要全面对话伙伴，双方通过专门的全面伙伴关系框架开展高级别对话。此外，东盟外长会批准接纳德国与卡塔尔成为东盟新的“部门对话伙伴”。</p>
<p>面对南海问题与缅甸局势等严峻安全挑战。东盟重申根据国际法保障航行和飞越自由的重要性，承诺尽快推进关于《南海行为准则》（COC）的谈判；针对缅甸危机，东盟致力落实关于停火、开展对话以及提供人道主义援助的“五点共识”，且开始探索灵活途径促进缅甸民族和解。</p>
<p>东盟外长们关切中东紧张局势，呼吁各方保持克制，强调霍尔木兹海峡航行安全。呼吁遵循国际法与联合国宪章，通过多边主义化解冲突。 </p>
<p>东盟坚持“东盟中心地位”，立足维护自身的团结与独立性，寻求在亚太、印太、乃至全球区域日趋复杂的大国政治博弈中避免沦为大国博弈的战场。</p>
<ul> <li>东盟-重要伙伴国对话</li> </ul>
<p>此次会议期间同步举行东盟与对话伙伴国（10+1）、东盟与对话伙伴国（10+3）外长会议。东盟成员国外长分别与中国、美国、俄罗斯、日本、韩国、印度等核心对话伙伴国举行了多场多边及双边高级别会谈。议题分别聚焦区域安全、经贸合作以及科技创新。</p>
<ul> <li>东盟-中国贸易与安全双线演变</li> </ul>
<p>东盟与中国外长对话聚焦强化人工智能（AI）领域合作，双方达成重要共识，将2027年确定为“东盟-中国人工智能合作年”，重点深化数字经济与创新领域的合作。</p>
<p>中国作为东盟最大贸易伙伴，东盟与中国经贸全面升级。不过持续升温的南海情势却让东盟内部团结亮起了分裂的警示灯。中国坚持把控南海主导权，反对引入外部势力干预。恰逢菲律宾主办东盟外长会议，由于各国对华立场不一，这对东盟机制内部的团结构成了重大挑战。</p>
<p>目前中国和菲律宾在南海对峙。南海局势仍处于“对抗性谈判”状态，各方均不愿在主权问题上做出妥协。与此同时，东盟与中国就南海行为准则进行的谈判进展缓慢，缺乏有效动力。</p>
<p>据了解，东盟内部在南海问题上的利益诉求依然存在差异。作为东道主的菲律宾和强硬争取南海主权的越南，正不断强化与美、日等外部力量的安全绑定以抗衡中方压力。而以柬埔寨为代表的部分东盟成员国，则倾向于支持中方的领土主张或反对在东盟联合声明中过度向中方实施集体施压，以避免东盟平台变成彻底对抗中国的双边冲突角力场。东盟陷入两难境地---既要维持与中国的经济联系，又要维护与美国之间的国际法律秩序。一旦内部分歧处理不当，可能形成阻碍合作的外交僵局。</p>
<ul> <li>东盟-日本持续深化防务合作</li> </ul>
<p>东盟与日本外长会谈围绕推进“东盟—日本全面战略伙伴关系”及区域安全防线展开了深层磋商。这场会议由日本外相茂木敏充（Toshimitsu Motegi ）与东盟-日本关系协调国新加坡外交部长维文（Vivian Balakrishnan）共同主持，日方高调提出将在“政府安全保障能力强化支援”（OSA）框架下强化与东盟国家的防务绑定与合作。日相承诺在防卫装备、AI及关键矿产供应链安全等关键领域，加大对东盟国家的实质性资金与技术支援。</p>
<p>日本在东南亚地区常年深耕，致力经营不同于中美大国的深层影响力。除了继续巩固供应链安全，包括关键矿产、数字基础建设、半导体及能源技术与绿色合作。日本外长还出席了第17届湄公河-日本外长会议，聚焦特定区域合作，深化水资源管理、跨境犯罪打击及数字人才培养，旨在增强湄公河流域的自主发展能力。</p>
<ul> <li>东盟-俄罗斯深化战略合作</li> </ul>
<p>东盟与俄罗斯外长会重点锁定粮食、能源供应链安全。为了因应俄乌冲突对全球供应链的持续冲击，双方落实了关于战略燃料储存与液化天然气贸易的务实合作。双方还同意在反恐、跨国犯罪和数字安全技术上加强协作。</p>
<p>这无疑是协助俄罗斯在西方全方位制裁下稳固并拓展其在东南亚的外交与经济空间实施地缘突围。毕竟对俄罗斯而言，此次东盟多边舞台是其拓展战略外交空间、打破西方全方位孤立的关键阵地。</p>
<p>今天登场的美俄外长会晤备受关注。美国国务卿卢比奥和俄罗斯外长拉夫罗夫（Sergey Lavrov）将围绕结束乌克兰战争的潜在途径展开讨论。尽管希望渺茫，但这是美俄两国围绕乌克兰和平谈判进行试探外交的一次重要突破。分析指出，关于结束俄乌战争的谈判似乎陷入停滞，原因是华盛顿转向关注针对伊朗的战争。</p>
<p>俄乌战争虽然发生在欧洲，但其长期的全球连带效应直接伤害到了东南亚各国的切身利益，尤其粮食与能源安全在东盟外长会上被反复讨论。东盟国家极力避免在乌克兰危机引发的“阵营对抗”中被迫选边站队。东盟外长在联合声明中重申维护国际法、联合国宪章以及和平解决争端的重要性，试图在战火引发的国际秩序撕裂中，保护东南亚自身的战略自主空间。</p>
<ul> <li>东盟-印度确定2026海上合作年</li> </ul>
<p>东盟-印度外长对话由东盟-印度关系协调国菲律宾与印度共同主持。双方围绕推进“东盟—印度全面战略伙伴关系（CSP）”展开了深度磋商。聚焦确立“2026东盟-印度海上合作年”，推进海洋安全治理，与东盟多国开展海洋安全对话。印度外交部长苏杰生（S. Jaishankar）在会上发言明确强调了海洋规则的重要性，越南等东盟国家予以支持。</p>
<p>印度目前是东盟第8大贸易伙伴和第9大投资来源国。双方致力于提升应对外部经济波动的坚韧性，拟启动自贸协定（AITIGA）谈判。苏杰生指出，印度与东盟两大经济体人口总计高达20亿，在世界过渡期必须通过深化合作来分散风险，共同携手破浪前行。</p>
<p>印度外长苏杰生还出席了“四边安全机制”（Quad）外长会。苏杰生与美、日、澳外长在场边单独开会，发表了旨在加强海洋安全、提升对东盟国家能力支援的联合文件，旨在平衡中国在该地区日渐增强的影响力。</p>
<p>中印外长会晤时，中国外长王毅向苏杰生表示中印应巩固双边关系向好势头、妥善处理分歧与敏感问题。苏杰生则强调尊重中方主权，并感谢中方支持印度办好2026年度金砖峰会。</p>
<ul> <li>多场重头戏马尼拉登场</li> </ul>
<p>东盟与中日韩（10+3）外长会今天正式召开，东盟11个成员国与中国外交部长王毅、日本外相茂木敏充（Toshimitsu Motegi ）、韩国外交部长官赵显（Cho Hyeon）共同出席，核心议题是推进东盟与中日韩三国在财经、贸易连接、粮食安全、产业链等传统领域的“10+3”区域合作，多国代表还拟围绕区域能源安全、关键矿产供应链、携手打击跨国网络诈骗以及当前复杂的国际与地缘政治局势提出讨论。</p>
<p>除10+3会议外，马尼拉会场还接连举行了东亚峰会外长会（EAS）和东盟地区论坛（ARF）部长级会议，各方正全力就中东局势升级带来的原油及航道安全冲击商讨集体应对方案。东亚峰会外长会由于包含中、美、日、俄、韩、朝等所有亚太安全核心博弈方，使得它不仅仅是一场部长级的正式多边会议，更是观察大国地缘关系的风向标。</p>
<ul> <li>大国边场外交引领地缘政治风向</li> </ul>
<p>综合东南亚媒体报道，在马尼拉举行第59届东盟系列外长会议期间，多国政要悉数到场且在场边开展双边或多边外交（Sideline Diplomacy），随即成为外界关注焦点，评论称，超级大国开展边线外交的激烈程度与外交含金量甚至超越了东盟框架下的主题会议。</p>
<p>中美外长周三(7月22日)举行90分钟闭门会谈，媒体盛传为中国领导人习近平9月访问美国铺垫基础。美国国务卿卢比奥（Marco Rubio）透露双方还讨论了中国在南中国海和台湾海峡的行为。中方则对美国对台湾价值140亿美元的军售议题表示关切。</p>
<p>美国、日本和韩国举行了三边会晤。美日韩外长在场边举行了密集的闭门会谈，美国国务卿卢比奥（Marco Rubio）重申了对地区盟友的安全承诺，并与日韩共同就朝鲜半岛局势、台海和平及网络安全防范达成一致。</p>
<p>中印外长相互释放向好讯号，中国与东盟侧重人工智能合作并寻求在和平稳定中进一步发展经贸合作，然而在复杂演变的地缘政治角逐中，各方都结合自身利益借机开展密集的双边和三边外交斡旋，试图在急剧变化的地缘政治局势中，谋求相对稳定的地位与角色。其中最具突破性且引起外界瞩目的正是今日上午展开的美俄场边高层会晤。</p>
<p>迄今为止，俄乌战争能否结束尚且是个未知数，而更值得注意的是俄罗斯外交部长拉夫罗夫的最新表态，他指出在朝鲜遭到联合施压的情况下，俄罗斯不再支持确保朝鲜半岛无核化的呼吁。此前朝鲜领导人金正恩声称将为海军配备核武器，并建造更大的军舰。</p>
