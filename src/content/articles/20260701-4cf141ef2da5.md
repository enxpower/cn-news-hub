<h2>我的阅读报告</h2>
<h3>睡觉很重要</h3>
<img src="https://cdnfile.sspai.com/2026/05/19/article/deb37d5da3635d4d8572bb065cb1c229.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>概览的书摘来自去年我很喜欢的一本书《我们为什么要睡觉》，前几年的成功学总是渲染很多高精力人群仿佛「不需要」睡觉的成功日程表，这本书就是想告诉我们，睡眠这么多年都没有进化掉，就是因为睡眠相当重要，不仅与我们的大脑相关，也与我们的情绪相关。</p>
<p>别说人要睡觉了，现在 AI 也需要「睡觉」了，Claude Code 和 Qwen Code 都推出了 dream 功能，让 Agent 可以通过类似人类睡眠期间整合记忆的办法，整合上下文记忆，然后更聪明工作。你就说睡觉重不重要吧。</p>
<h3>周中爱阅读</h3>
<img src="https://cdnfile.sspai.com/2026/05/19/article/72b4c7c8f5b442339ce90fd0a041201a.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>时间节律方面，我发现去年的阅读时长整体比较长，很正常，因为去年有段时间我没在工作，就狂看书。我的月读书天数则很不稳定，为什么呢？大概因为低的那些时间我在看纸质书。总体也是很准确了。</p>
<p>阅读日的时长比较集中在 30-60 分钟之间，要么我就不打开微信读书了，那真的很取决于书好不好看，以及我的空闲时间，很神奇的是我反而周中看书更多，可能是规律的生活让我的阅读也很有规律，周末都在忙着疯玩吧。</p>
<img src="https://cdnfile.sspai.com/2026/05/19/article/a94bf39220201b4e15ccef45db843c58.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>2024 年微信读书打开得更多，但均阅读时间少，2025 年是只要打开了微信读书，大概就会读 1 小时左右。今年也差不多，这算是很有意思的发现。</p>

<p><strong>文学的阅读量简直一骑绝尘啊</strong>！读社会纪实累了，读本小说奖励一下自己，这大概最能概括我的阅读风格了，我对文学的热爱已经到达了水滴形💧的程度，其次我喜欢社会纪实方向的，也包括各种人物传记。然后就是女性小说。</p>
<p>很合理了，我很在意真实世界、女性写作和女性经验。</p>

<p>我阅读最长的作者们有五位女性作者，很满意了！最喜欢的出版社是上海译文出版社，大概是因为译文纪实系列是我很喜欢的书系，《东京贫困女子》《世上为什么要有图书馆》《致命爱人》和《凶年》都是这个系列之一。</p>

<h2>跟自己的书架对话</h2>
<p>通过微信读书 Skill，你不仅可以使用技能输出自己的阅读报告分析，也可以直接在 Qwen Code 中直接跟微信读书书架对话：</p>
<p>我在《纷乱的心灵》里划了 135 条线，在《空洞的心》里划了 106 条。两本都是加博尔·马泰写的，讲成瘾和注意力缺陷。平时这些划线就躺在那里，现在可以直接问。</p>
<p>不仅如此，你还可以使用这个技能写文章找素材：</p>
<blockquote><p>我想写一篇关于「成瘾与自我认知」的文章，从我的笔记里找出所有相关的划线和想法</p></blockquote>
<p>还可以看看最近在想什么：</p>
<blockquote><p>导出我最近一个月所有书的想法和点评，帮我看看最近在关注什么主题</p></blockquote>
<p>写书评：</p>
<blockquote><p>我刚读完《绿灯》，结合我在这本书里的 35 条笔记，帮我写一篇 500 字的短书评</p></blockquote>
<p>社交分享：</p>
<blockquote><p>从我最近读完的 3 本书里，各挑一条划线，帮我写一条适合发朋友圈的阅读分享</p></blockquote>
<p>知识卡片：</p>
<blockquote><p>把我在《空洞的心》里关于成瘾的核心划线，做成 5 张知识卡片的文案，每张一个观点加原文引用</p></blockquote>
<p>生成 Obsidian 读书笔记：</p>
<blockquote><p>把我在《纷乱的心灵》里的划线和想法，整理成一篇 Obsidian 格式的读书笔记，保存到当前目录。包含 frontmatter、标签、按章节组织划线。</p></blockquote>
<p>微信读书 Skills + Qwen Code + Obsidian 能搭起一条阅读到知识库的流水线。大概是这样：</p>
<blockquote><p>微信读书（数据源）→ Qwen Code（拉数据 + AI 处理）→ Obsidian（本地知识库）</p></blockquote>
<p>它能做到每本书自动生成一个笔记文件、AI 自动打标签和双向链接、按主题生成 MOC 索引页、新划线增量合并到已有笔记里。非常适合我的图书馆建设——我之前都是手动复制笔记（说多了都是泪）。</p>

<h2>如何使用？</h2>
<p>微信读书 Skills 支持 Qwen Code、Claude Code、Cursor 这些工具都行。我主要使用 Qwen Code，另外它跑在终端里，能直接帮你写文件、跑脚本。生成 Obsidian 笔记、跑可视化报告都是一句话。</p>

<h2>小结</h2>
<p>我很喜欢逛的一个博主，他的网站上有一句话我记了很久，「一个人的思想史就是 ta 的阅读史」。</p>
<p>阅读的变化能让你看到自己的变换与思考的脉络，就像前几年更喜欢政治哲学的作品，这两年更爱纪实和心理，也反映出我关注重点的变化。我写了好几次，「我认为 AI 是人类很好的镜像」，你使用 AI 的方式，也会反应你思考自身的方式，它是作为伙伴，还是作为指导者，它的位置取决于你对它的判断和对自我的判断， AI 的思考过程是收敛的，它会逐渐聚焦，而人类的思考是发散的，会从苹果联想到手机，联想到乔布斯，联想到科技与人文的交叉，然后想到牛顿最后投身于神学，想到宗教对人类社会的影响。</p>
<p>这个联想的过程正是思考的乐趣所在。阅读是建立这些在当前看来「不可知」的联想的基础。</p>
<p>我跟过往一样热情拥抱微信读书推出的 Skill，因为我相信 AI 会成为逐渐明晰自己的思考和辩论的伙伴，毕竟人对自己或许会偶尔不诚实，但你的书架是很诚实的。</p>
