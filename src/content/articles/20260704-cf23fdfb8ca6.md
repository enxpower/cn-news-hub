<p>7月4日，据追风交易台消息，高盛7月3日发布最新研报称，MiniMax电话会释放了强烈的商业化与技术演进信号：<strong>管理层对2026年底实现10亿美元的年度经常性收入（ARR）目标充满信心。</strong></p>
<p>报告称，最核心的催化剂在于<strong>中国AI大模型行业正迎来“价格战”的拐点</strong>——随着竞争对手DeepSeek宣布高峰期涨价，行业定价正回归理性。</p>
<blockquote>
<p>MiniMax凭借<strong>超过90%的算力利用率、与国产芯片的高度集成以及独特的“组织敏捷性”</strong>，在维持极具竞争力的定价（<strong>M3模型混合价格每百万Token 0.22美元</strong>）的同时，实现了远超同行的毛利率。此外，<strong>即将于数周内推出的H3视频生成模型将进一步打开多模态市场的想象空间。</strong></p>
</blockquote>
<p>高盛维持<strong>买入评级</strong>，12个月目标价<strong>港币860元</strong>，较当前股价港币356.80元隐含<strong>141%上行空间</strong>。</p>
<img src="https://wpimg-wscn.awtmt.com/1ade5825-67a0-443f-a6ac-d18d1d8da8a9.png?imageView2/2/w/680" alt="" loading="lazy">
<h2>ARR增长轨迹：从1亿到10亿，管理层给出明确路径</h2>
<p>报告称，MiniMax管理层在电话会上系统性地梳理了ARR（年化经常性收入）的增长节点：</p>
<blockquote>
<ul>
<li>
<p><strong>2025年12月底</strong>：ARR达1亿美元；</p>
</li>
<li>
<p><strong>2026年2月</strong>：ARR升至1.5亿美元；</p>
</li>
<li>
<p><strong>2026年4月</strong>：ARR较2月再度翻倍；</p>
</li>
<li>
<p><strong>M3模型于6月1日正式上线前</strong>：ARR进一步加速爬升。</p>
</li>
</ul>
</blockquote>
<ul>
<li>
<p><strong>2025年12月底</strong>：ARR达1亿美元；</p>
</li>
<li>
<p><strong>2026年2月</strong>：ARR升至1.5亿美元；</p>
</li>
<li>
<p><strong>2026年4月</strong>：ARR较2月再度翻倍；</p>
</li>
<li>
<p><strong>M3模型于6月1日正式上线前</strong>：ARR进一步加速爬升。</p>
</li>
</ul>
<p>管理层明确表示，对<strong>2026年底实现10亿美元ARR目标</strong>保持充分信心。</p>
<p>在定价策略上，M3与上一代M2.7保持相同定价，但管理层强调这一策略在毛利率层面具有可持续性——原因在于训练与推理架构升级带来<strong>超过2倍的成本节省</strong>，基本抵消了总参数量翻倍所带来的成本增加。</p>
<p>公司还预告，<strong>2026年下半年将推出更大规模的M3系列模型，目标是在保持强劲成本性能的同时进一步提升智能水平。</strong></p>
<p>这一ARR增长曲线是支撑高盛收入预测的核心依据——高盛预计MiniMax收入将从2025年的7900万美元跃升至2026年的3亿美元，2027年进一步达到8.801亿美元，2028年突破24.696亿美元。</p>
<h2>DeepSeek涨价：行业定价理性化的破冰信号，直接利好MiniMax</h2>
<p>这是高盛报告中最具市场催化意义的外部事件。</p>
<p>DeepSeek本周宣布，其V4正式版将于7月中旬上线，同步引入<strong>高峰/非高峰API差异定价机制</strong>：高峰时段（北京时间上午9点至12点、下午2点至6点）收费为非高峰时段的<strong>2倍</strong>，混合定价约为每百万token <strong>0.35美元（Pro版）/ 0.12美元（Flash版）</strong>。</p>
<p>高盛将此解读为：<strong>自2026年4月底以来中国AI模型公司激进定价（部分玩家毛利率为零甚至为负）正在向更理性阶段过渡的早期信号</strong>，本质上是推理成本压力在定价端的真实反映。</p>
<p>对比之下，MiniMax M3的混合定价为每百万token <strong>0.22美元</strong>，在性能/价格比维度上具备显著竞争优势，且毛利率明显高于同业——这得益于其更高比例的自建优化算力，以及以更小激活参数实现高效推理的架构设计。</p>
<p>MiniMax还特别指出，其自营算力可实现<strong>超过90%的利用率</strong>，通过高峰时段服务知识工作者和开发者、低谷时段将闲置算力用于实验和数据整理的方式，实现算力的峰谷平衡调度，从而支撑长时程智能体工作流的成本优势。</p>
<h2>H3视频模型：数周内发布，深度融合M3架构</h2>
<p>与此同时，MiniMax即将推出下一代视频生成模型<strong>H3</strong>，预计在"未来数周内"正式发布。</p>
<p>H3的核心升级体现在两个维度：</p>
<ul>
<li>
<p><strong>视频生成质量与功能多样性</strong>的全面提升，背后是重大架构升级（包括标注/分类/反馈循环的优化）；</p>
</li>
<li>
<p><strong>与M3模型架构的深度整合</strong>：大语言模型能力被嵌入H3的DiT（扩散变换器）架构中，例如增强对人类动作和基本物理关系的理解能力。</p>
</li>
</ul>
<p><strong>视频生成质量与功能多样性</strong>的全面提升，背后是重大架构升级（包括标注/分类/反馈循环的优化）；</p>
<p><strong>与M3模型架构的深度整合</strong>：大语言模型能力被嵌入H3的DiT（扩散变换器）架构中，例如增强对人类动作和基本物理关系的理解能力。</p>
<p>此外，MiniMax正在引入垂直领域专家，逐步切入<strong>长片电影/剧集制作市场</strong>，拓展视频生成的商业化边界。</p>
