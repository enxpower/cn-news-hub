小米开源了其 AI 编程助手 MiMo Code，源代码采用 MIT License 托管在 GitHub 上。小米博客称，“MiMo Code 是小米 MiMo 团队基于 OpenCode 构建的终端编程 Agent，MIT 协议开源。它针对长程自动化编程任务设计，核心关注点是：如何在几十甚至上百步的持续执行中保持决策质量和状态连续性。”
