<h2><strong>什么是CiGA Game Jam？</strong></h2>
<p>Game Jam 是一种特定的非赛事开发活动形式，在 Game Jam 中开发者在规定的时间内（通常为48小时），根据现场给出的主题，完成一款包含核心玩法的游戏 Demo。对于线下的 Game Jam 活动来说，与现场第一次认识的开发者们临时组团结识更多伙伴，策划、美术、程序、音效通力合作，也是 Game Jam 活动的乐趣之一。</p>
<p>Game Jam&nbsp;活动很多，而 CiGA Game Jam 则是华语圈最大的线下 Game Jam 活动，今年是该活动的第 12 个年头。去年 CiGA Game Jam 共有 2000 多名开发者参与其中，最终在 48 小时内完成了 450 多款游戏原型。</p>
<img src="https://cdnfile.sspai.com/2026/06/12/82a8ab18cb39a874208b6ab4f6807eec.png?imageView2/2/w/1120/q/90/interlace/1/ignore-error/1" alt="" loading="lazy">
<p>二月的时候，少数派广州线下店首次参加 Global Game Jam，成为了 2026 年次的指定站点之一。七月的 CiGA Game Jam 我们当然也不想错过。欢迎游戏开发者们再次光临～</p>

<h2><strong>关于站点发起人</strong></h2>

<h2><strong>关于站点场地信息</strong></h2>
<p>少数派广州线下体验店于 2025&nbsp;年初正式开业，坐落于荔湾湖畔泮塘五约古村，延续其 “高效工作、品质生活”&nbsp;的核心理念，打造集数码、音乐、社区于一体的复合型线下据点。店内一侧为少数派自研与联名数码好物，标志性的 K3&nbsp;系列机械键盘、定制开箱刀、水獭 IP&nbsp;公仔等；另一侧为黑胶、CD&nbsp;等唱片体验区，配备了飞傲、真力、Austrian Audio&nbsp;等专业音频设备。</p>
<p>店里不定期举办音乐沙龙、唱片展等活动，延续着少数派对创作者的长期关注。作为线上社区的线下延伸，这里既是数码爱好者交流体验、打卡少数派文化的实体据点，也是老广文艺青年逛湖漫步、放空小聚的隐秘角落。隔壁的 Heim&nbsp;精酿酒吧也是店铺的一部分，里面也有着丰富的设备和唱片，两者无缝联动，形成 “数码 +&nbsp;音乐 +&nbsp;精酿 +&nbsp;老城烟火”&nbsp;的独特生活方式，逐渐吸引到了越来越多的年轻人。</p>
