<h2 class="text-[#333] text-[15px] font-semibold mb-2" data-svelte-h="svelte-iuetu3">核心要点</h2>
<ul class="list-disc pl-5 text-[#333] text-[15px] leading-[26px]"><li>只买韩国AI概念股，已经装不下韩国散户的上车焦虑</li><li>从MiniMax到澜起科技、北方华创，再到机器人ETF和电网设备ETF，韩国散户买入的中国资产逐渐沿着AI产业链铺开</li></ul>
<p>这轮投资的数字很难不让人侧目。韩国总统李在明宣布，韩国要围绕半导体、物理人工智能和数据中心推进“三大轴心”。三星电子和SK海力士将在韩国西南部各建两座大型半导体工厂，相关投资规模达到800万亿韩元，折合超过5000亿美元。</p>
<p>韩国政府还计划在忠清地区推进芯片封装集群，在光州、全罗南道等地发展新的半导体产业基地。韩国科技部门提出，到2035年，人工智能数据中心相关投资可能超过1000万亿韩元（约6500亿美元）。</p>
<p>人工智能热度也越过韩国本土市场，钻进了韩国散户的海外账户。</p>
<p>今年1月，大模型厂商MiniMax在港交所挂牌。一个多月后，韩国预托结算院数据被媒体翻出：MiniMax自上市以来吸引韩国散户净买入约2100万美元；澜起科技获得约1900万美元净买入；内地市场里，北方华创获得约350万美元净买入。</p>
<p><strong>一位受访韩国散户早先接受媒体采访时说，自己买入MiniMax，是押注这家“中国版OpenAI”的股价上涨。他同时持有20多只全球科技股。</strong></p>
<p>不过，就在最近，这场“下半场”突然出现了急转弯。</p>
<p>7月2日，韩国芯片股遭遇抛售，SK海力士、三星电子大跌，年内一度涨超10倍的南方两倍做多海力士ETF，日内跌幅超过20%。</p>
<p>几天前，它还是韩国AI市场狂热的放大器，不断地挑动着投资者的神经。</p>
<p>韩国网络里，一个流行词在今年重新出现：“벼락거지”，翻译过来是“突然成了穷人”。它曾在房地产上涨时期流行，用来形容资产价格快速上涨后，没赶上车的人突然产生相对贫困感。</p>
<p>如今，高点接盘的股民，似乎符合“一夜变穷”形容。</p>
<p>变穷的焦虑和暴富的热情，交织在韩国股市内外。<span data-pm-slice="1 1 [&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;data-role&quot;:&quot;outer&quot;,&quot;label&quot;:&quot;edit by 135editor&quot;,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;data-role&quot;:&quot;outer&quot;,&quot;label&quot;:&quot;edit by 135editor&quot;,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;style&quot;:null,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;p&quot;,&quot;attributes&quot;:{&quot;style&quot;:&quot;margin-left: 8px; margin-right: 8px; line-height: 1.75em;&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;}]"><br></span></p>
<h2><span data-pm-slice="1 1 [&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;data-role&quot;:&quot;outer&quot;,&quot;label&quot;:&quot;edit by 135editor&quot;,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;data-role&quot;:&quot;outer&quot;,&quot;label&quot;:&quot;edit by 135editor&quot;,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;section&quot;,&quot;attributes&quot;:{&quot;style&quot;:null,&quot;data-pm-slice&quot;:&quot;0 0 []&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;},&quot;para&quot;,{&quot;tagName&quot;:&quot;p&quot;,&quot;attributes&quot;:{&quot;style&quot;:&quot;margin-left: 8px; margin-right: 8px; line-height: 1.75em;&quot;},&quot;namespaceURI&quot;:&quot;http://www.w3.org/1999/xhtml&quot;}]">01</span></h2>
<p>MiniMax进入韩国散户买入榜，标记出韩国人工智能投资热的一次外溢。</p>
<p>在韩国内部，三星电子和SK海力士，已经是全球AI基础设施里最醒目的韩国名字。高带宽内存（HBM，人工智能芯片常用的高性能内存）需求暴涨后，SK海力士的股价、市值都水涨船高。</p>
<p>但韩国散户的账户没有停在本土芯片股上，海外的大模型厂商也引发诸多关注。</p>
<p>今年1月，MiniMax在港交所挂牌，发行价165港元，上市首日股价大涨，募资约6.2亿美元。一家中国公司，却很快变成韩国散户眼里的人工智能新标的。</p>
<p><strong>一个多月后，韩国预托结算院数据显示，MiniMax获得韩国散户约2100万美元净买入。</strong></p>
<p>同一批数据里，韩国投资者年内买入约9250万美元香港上市人工智能和科技股。MiniMax、华夏沪深300交易型ETF、澜起科技排在前列，恒生科技ETF、英诺赛科、中国科创50ETF、中国半导体ETF也在名单中。</p>
<p>韩国散户买入的范围，从大模型公司，延伸到半导体、科技指数和硬件链条。</p>
<p>而在MiniMax之后，韩国资金还在加速寻找中国科技主题资产。华夏机器人ETF获得韩国资金321万美元净买入，电网设备主题ETF获得超过110万美元买入。机器人、电网设备和半导体一起进入韩国散户的中国资产清单。</p>
<p>这也从侧面说明了韩国投资者的狂热：只要是能和人工智能沾边的产业链，就会有大量韩国投资者涌入。</p>
<p>上个月开始，韩国投资者开始涌入中国AI芯片市场。SEIBro数据显示，过去一个月，寒武纪成为韩国投资者净买入最多的A股标的，单月净买入接近2000万美元，约合人民币1.3亿元。</p>
<p><strong>韩国投资者近一个月净买入前十的A股里，还有麦格米特、机器人ETF华夏、宁德时代等公司。中国的半导体和具身智能企业，都成了韩国股民的优先“扫货”选择。</strong></p>
<p>面对投资者的踊跃热情，韩国本土金融机构也在推波助澜。今年3月，KB资产管理在韩国交易所上架RISE中国AI半导体Top 4 Plus ETF，底层指数覆盖中国内地和香港15家AI半导体公司，范围包括半导体代工、AI芯片、光通信模块、材料与设备。</p>
<p>收益率很快给了韩国投资者反馈。韩国市场上另一只KODEX中国AI半导体TOP10ETF，今年上市后一个月上涨31.81%。这种趋势一直持续到6月下旬，中国主题产品在韩国ETF收益榜上名列前茅。</p>
<p>从MiniMax到澜起科技、北方华创，再到机器人ETF和电网设备ETF，韩国散户买入的中国资产逐渐沿着AI产业链铺开。</p>
<p>韩国本土已有三星电子、SK海力士这样的AI芯片标的。但买入名单继续向中国AI新股、半导体链条、先进制造主题延伸，说明这轮AI交易还在膨胀。</p>
<p>只买韩国AI概念股，已经装不下韩国散户的上车焦虑。</p>
<h2>02</h2>
<p>今年5月，SK海力士市值突破1万亿美元。高端内存芯片价格一季度翻倍，二季度预计继续上涨。SK海力士在全球高带宽内存市场份额约58%。SK集团会长崔泰源称，公司计划未来五年将晶圆产能翻倍。</p>
<p><strong>这已经足够刺激普通投资者的神经，但是资本市场的风起云涌还没有停止。</strong></p>
<p>上周，韩国总统李在明宣布，要把半导体、物理人工智能、数据中心一起做成国家工程。三星电子和SK海力士将在西南部各建两座大型半导体工厂，相关投资规模达到800万亿韩元。</p>
<p>韩国6月出口数据也给这轮故事添了一把火。韩国当月出口同比增长70.9%，达到1022.5亿美元，创下近半个世纪以来最快增速。其中，半导体出口同比增长199.5%，达到448亿美元。</p>
<p>人工智能投资还带动电脑销售增长308.8%，数据中心建设带动钢铁产品出口增长。</p>
<p>产业的指数级盈利增长，落到普通人身上就是一个个“造富”神话。</p>
<p>有媒体报道提到，韩国婚恋机构已经开始重新评估半导体员工的吸引力。过去，医生、律师、富裕家庭出身者更容易被列为“A+候选人”。</p>
<p><strong>今年，三星电子和SK海力士员工开始成为了相亲市场的不二之选。一位婚恋顾问说，介绍SK海力士员工时，常常引发对面难以置信的惊叹：这样的人也在这里吗？</strong></p>
<p>三星和SK海力士的奖金也成了韩国社会的八卦素材。报道提到，三星芯片部门部分员工可能拿到约41.6万美元的绩效奖金。要知道，韩国工人2024年平均年收入约4500万韩元，折合不到3万美元。</p>
<p>这个差距足以制造新的社会想象。</p>
<p>过去在韩国，医生、律师、检察官、首尔核心区房主，是最稳定的上升通道。人工智能芯片热起来后，另一条通道突然挤进来：去三星电子，去SK海力士，去半导体工厂，去会发奖金的地方。</p>
<p>报道称，一些学生开始关注职业学校，因为部分半导体工厂岗位并不要求大学学历。补习机构也出现了帮助应聘三星、SK海力士的课程。</p>
<p><strong>也正是在这种时候，“벼락거지”——“一夜变穷人”的概念重新流行。</strong></p>
<p>这个词曾在房地产上涨时期流行，用来形容资产价格快速上涨后，没赶上车的人突然觉得自己被甩下。</p>
<p>今年，韩国网友把它放进人工智能语境。社交媒体和网络社区里，有人说“只存款，好像落后了”，也有人表示“感觉只有我没变富”。</p>
<p>《韩国经济新闻》作者赵镇亨今年4月写下《AI一夜变穷人的愤怒》。他写到人工智能法律意见书，也写到有人用Claude开发角色扮演游戏。</p>
<p>真正给大众冲击的，是4月连续出现的一组数字：三星电子季度收入、SK海力士利润率，以及韩国股市大盘持续上行。</p>
<p>人工智能最早带来的震动，可能是工作会不会被替代。到了韩国股市这轮行情里，震动变成了另一个问题：别人是不是已经靠人工智能变富。</p>
<p>这种焦虑很快进入基金销售现场。</p>
<p><strong>今年5月，韩国国家成长基金开售。它是李在明政府推动“生产性金融”的一部分，目标是把家庭储蓄从存款和房地产，引向人工智能、半导体、生物、机器人、二次电池等战略产业。面向个人投资者的年度额度是6000亿韩元。</strong></p>
<p>首尔新韩银行总部主支行贴出“线上额度售罄、线下额度售罄”的公告。部分线上平台从早上8点开售后，大约10分钟卖完。几天后，面向个人投资者的额度几乎售罄。</p>
<p>对散户来说，这种机会很难旁观。“一夜变穷人”再度流行时，韩国散户已经在排队争抢下一只AI基金。</p>
<h2>03</h2>
<p>资本市场疯狂拥簇AI的热度之外，韩国人工智能投资热最锋利的争议，落在两块屏幕上。</p>
<p>一块是股市屏幕。韩国综合股价指数（KOSPI，韩国股市主要大盘指数）今年以来不断被人工智能芯片股推高。</p>
<p><strong>另一块是外汇屏幕。韩元兑美元此前跌至两个月低点附近，盘中逼近1520。首尔财经媒体提到，当时美元指数并不强，欧元、英镑等主要货币走强，韩元表现却落后。</strong></p>
<p>一边是股市在庆祝，一边是韩元汇率在泼冷水。</p>
<p>本月初，最新的市场数据又补了一刀：今年上半年，外资从亚洲股市撤出约1373.6亿美元，是至少16年来最快速度。其中，韩国流出约708亿美元，是区域内最重的流出市场之一。</p>
<p>至于为什么撤出，相关分析显示，韩国和中国台湾的人工智能芯片股涨得太快，全球基金里的相关仓位被动变重，基金经理不得不卖出部分赢家，重新平衡组合。</p>
<p>换句话说，韩国芯片股涨得太快、太猛，让外资必须减仓。</p>
<p>而在同一时间，韩国散户还在买，还在抢，还在加杠杆。人工智能行情越涨，错过感越强；外资越卖，散户越像站在另一边接筹码的人。</p>
<p><strong>这种分裂，很快被杠杆产品放大。</strong></p>
<p>今年5月，韩国首批挂钩三星电子、SK海力士的单股杠杆及反向交易型开放式指数基金上市，共16只。所谓“二倍三星”“二倍海力士”，就是把三星电子、SK海力士单日涨跌放大到两倍。涨的时候更快，跌的时候也更快。</p>
<p>这类需求一开始非常热，一些媒体报道提到，韩国金融投资协会要求散户完成培训才能交易这类产品，第一天网站就因为申请量太大而崩溃。此后超过35万人完成课程。</p>
<p>人工智能浪潮中，散户不仅要研究财报、看芯片价格，还要去上课，拿到交易高风险产品的资格。仿佛只要慢一步，就会被下一轮行情甩下。</p>
<p>与此同时，美股市场上的Roundhill Memory ETF也成为了韩国股民参与投资的重要入口。</p>
<p><strong>这只基金主投全球记忆体芯片公司，覆盖高带宽内存、DRAM（电子设备常用的基础内存类型）等方向。它的底层持仓包括SK海力士、三星电子、Micron等记忆体公司。</strong></p>
<p>今年5月，Roundhill Memory ETF成为韩国海外投资散户净买入最多的美国ETF，净买入3.1715亿美元。它在短时间里快速扩张，成为记忆体芯片行情里最受追捧的工具之一。</p>
<p>这就是韩国人工智能交易的奇特循环：本土账户买三星和SK海力士，海外账户买美国记忆体ETF，港股账户买MiniMax和中国科技ETF。几个市场绕一圈，最终都在押同一个问题：人工智能基础设施还能涨多久。</p>
<p>除了现实，没有人能给出答案。</p>
<p>6月23日，韩国综合股价指数单日下跌9.99%，创下三个多月来最大跌幅。三星电子和SK海力士单日跌超12%，市场还触发20分钟交易暂停。触发因素之一，是韩国监管层对新推出的杠杆ETF发出过热警示，随后海外投资者抛售芯片股。</p>
<p><strong>韩国股民前几天还在讨论“二倍海力士”，转头就看见交易暂停。</strong></p>
<p>一位首尔居民此前用约1500万韩元短期透支额度买入三星电子杠杆基金，最初盈利约20%。随后韩国综合股价指数在科技股抛售中大跌，她的仓位转为亏损17%。</p>
<p>要知道，她买的不是冷门小票，而是三星电子杠杆产品。</p>
<p>韩国央行数据显示，截至今年5月，韩国散户股票杠杆投资超过60万亿韩元。韩国金融投资协会数据也显示，韩国综合股价指数市场散户借款投资达到29万亿韩元，较去年底增长71%。</p>
<p>韩国资本市场研究院院长金世完称，券商借款额度已经触顶。</p>
<p><strong>事实上，关于产业泡沫的讨论中，韩国AI产业是相对坐得住的一批企业。半导体出口暴涨，芯片厂继续扩建，三星和SK海力士确实站在全球人工智能基础设施的关键位置。</strong></p>
<p>但另一组数据也摆在那里：外资撤出、韩元承压，杠杆ETF被监管点名，股指单日暴跌近10%。到了本月，做多海力士ETF又单日跌超20%，把这轮AI交易的另一面摊开给散户看。</p>
<p>网络社区里，“一夜变穷人”的帖子还在更新。一涨一跌两组数据间，这场AI投资引发的投资焦虑，似乎并没有停歇的迹象。</p>
