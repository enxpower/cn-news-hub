<p>这是虎嗅WAIC“追踪Token商业新范式”系列文章第9期。</p>
<p label="正文">AI 的“活人感”越来越高了。</p>
<p>这意味着，我们越来越难分辨，一段视频究竟是真人拍摄，还是 AI 生成。</p>
<p>AI 视频正在以前所未有的速度进入内容生产环节。据相关数据显示，2026 年第一季度，全行业上线微短剧约12.8万部，其中 AI 微短剧约12.2万部，占比已经超过95%。从短剧、广告，到电商、游戏、影视制作，AI 视频正在逐渐成为一种新的生产工具。</p>
<p>在海外，一款名为 PixVerse 的 AI 视频产品，已经成为这一趋势的代表。用户只需要输入一段 Prompt，或上传一张图片，就能够在几十秒内生成一段高质量的视频。过去一年，这款产品迅速积累起全球用户，并成为 AI 视频领域增长最快的产品之一。据媒体报道，截至2025年底，PixVerse 母公司爱诗科技 ARR 已达到4000万美元。</p>
<p>虎嗅获悉，继2026年3月爱诗完成3亿美元融资后，爱诗科技于近日完成了29.8元人民币的C轮融资，由阿里巴巴领投，Lollapalooza Capital（王慧文家办）、常春藤资本、惠远资本、钟鼎资本、韩国未来资产、OCBC 生态下 Lion X 基金、蓝色光标、CloudAlpha、iGlobe Partners 等十余家国内外机构参与投资。</p>
<p>过去一年，整个 AI 行业都在讨论 Token。</p>
<p>最初，大模型时代的 Token，像是一个计算单位，它更多是在用文字来衡量token；随后，随着 Cursor、Claude Code 等 Coding Agent 的爆发，Token 又开始对应软件开发，越来越多开发者愿意为代码生成持续付费。而 AI 视频，则让 Token 第一次真正进入内容产业。</p>
<p>在 Coding 场景中，Token 对应的是代码，那么在 AI 视频时代，Token 对应的则是内容生产本身。一次模型推理，可以直接生成一条短视频、一段广告素材、一部 AI 短剧，甚至一个电商数字人。</p>
<p>更准确地说，AI视频时代，Token 对应的不只是模型消耗的算力，而是直接对应视频创作、影视制作乃至商业生产。</p>
<p>这也是为什么，AI 视频正在成为继 Coding 之后，最能验证 Token 商业范式的场景之一。</p>
<p>过去，创作一条视频是一件门槛极高的事情。一个视频创作者，需要完成脚本、拍摄、灯光、剪辑、特效、配音、后期等一整套流程，还需要熟练掌握 Premiere、After Effects 等专业软件。但在今天，这些复杂的创作流程则已经被压缩成一次模型推理，几句 Prompt 便可以完成过去需要数天甚至数周才能完成的工作。对于 AI 视频而言，每一次 Token 消耗，背后对应的是内容生产。</p>
<p>然而，这门生意远没有想象中那么容易。</p>
<p>一方面，AI 视频公司需要持续投入巨额算力成本；另一方面，它们还需要面对来自短视频平台、传统视频平台以及互联网巨头的竞争。目前，包括可灵、MiniMax旗下海螺AI等产品都已经积累了大量用户，整个 AI 视频行业正在快速进入竞争阶段。</p>
<p>更重要的是，在商业化路径上，行业也出现了明显分化。2026年，已经有一批视频生成模型公司涌了出来。</p>
<p>不少 AI 视频创业者都向虎嗅表示，相比 C 端市场，他们更愿意直接进入 ToB。有相关创业者告诉虎嗅，“ToC 市场虽然有更高的天花板，但同时也有大平台的竞争，很难切入。但ToB 的需求更明确，客户愿意为效率买单，ROI 更容易计算；而 ToC 不仅需要持续投入模型能力，还需要解决获客、留存、订阅以及产品体验等一系列问题，是一条更难走的路。”</p>
<p>但爱诗科技却并不这样想。在谢旭璋看来，AI 视频领域，C端消费者市场仍然是相当大的机会。</p>
<p>他认为，过去几十年，视频行业最大的商业机会始终来自 C 端。每一次内容产业的变革，最终诞生的都是新的消费者平台。而 AI 视频时代，同样应该诞生属于这个时代的新产品，而不仅仅是一家模型公司。</p>
<p>当然，爱诗科技不满足于仅涉足消费者市场。谢旭璋告诉我，“同时也有切入B端市场。”</p>
<p>围绕 AI 视频行业、Token 商业模式、ToC 与 ToB 的选择，以及 AI 视频未来的商业化方向，虎嗅与爱诗科技联合创始人谢旭璋进行了交流，以下为对话全文，有删减：</p>
<p label="Q">虎嗅：过去一年，爱诗最大的变化是什么？无论是技术、商业模式还是组织层面，都可以聊聊。</p>
<p label="A">谢旭璋：大的方向其实没有变。我们三年前创业时确定的技术路线和方向，到今天基本还是一致的。变化主要有三个。</p>
<p>我们过去更多聚焦海外市场，今年开始在海外建立本地团队，希望做更精细化的运营；公司团队规模从几十人扩张到100多人；</p>
<p>第三，也是最大的变化，我们过去主要做 ToC，今年也做了 ToB 探索，通过标准化 API 服务更多行业客户。目前 ToB 业务还处于早期阶段，但增长速度不错。</p>
<p>另外，我们越来越意识到，视频本身就是一个非常大的行业，里面有大量细分场景。我们自己做 ToC 产品，不可能覆盖所有行业，而很多行业更适合通过 API 的方式接入模型能力。随着整个 AI 视频市场不断扩大、参与者逐渐减少，我们认为 ToB 的空间也会越来越大。</p>
<p label="Q">虎嗅：不少 AI 视频创业公司都认为 ToC 很难，所以直接选择 ToB。但爱诗一直坚持 ToC，为什么？</p>
<p label="A">谢旭璋：我一直觉得，视频行业最大的市场会在消费者。</p>
<p>从很商业的第一性来看，视频行业过去几十年的发展已经证明了这一点。无论是电影院、长视频平台，还是后来出现的短视频平台，最大的机会一直都来自 C 端，而不是 ToB。</p>
<p>我们三年前创业的时候，就希望做最好的视频模型，同时进入最大的市场，所以坚定选择了 ToC。</p>
<p>过去几年，我们已经积累了不错的用户规模和收入，也因此看到了很多同行没有看到的机会。直到今天，我们做的很多事情依然属于非共识。另外，我并不认为视频行业会赢者通吃。</p>
<p>从电影时代，到长视频、短视频，每一次行业变革，都诞生了很多成功的公司，没有任何一个时代只有一家赢家。AI 视频也是一样。</p>
<p>我们认为，这个行业至少有三个机会。第一，AI 可以帮助从来没有做过视频的人完成人生中的第一条视频。第二，它会逐渐取代大量传统拍摄、3D 渲染等内容生产流程，真正进入影视、广告等行业。第三，随着 AI 原生创作者越来越多，还会诞生新的内容形态，以及人与内容新的交互方式。</p>
<p>所以，我们相信 AI 视频时代，同样会诞生新的平台，而不仅仅是模型公司。</p>
<p label="Q">虎嗅：所以，为什么现在又开始布局 ToB？</p>
<p label="A">谢旭璋：我们并不是战略转向，而是在通用模型成熟之后，自然开始进入更多行业。</p>
<p>我们的思路一直没有变，研发团队始终只做一件事，就是打造全球领先的通用视频生成模型。</p>
<p>在此基础上，我们会针对不同场景做优化。比如，在 ToC 产品里，我们更关注生成速度、交互体验，让普通用户更容易使用；在影视行业，我们会针对影视制作做模型微调；未来也会针对更多行业做类似的优化。</p>
<p>但底层模型始终是同一个，不会因为 ToC 或 ToB 而拆成两套研发体系。对于一些行业客户，我们也不会做完全定制化的模型。</p>
<p>如果一个需求只有一家客户能用，没有任何通用价值，我们一般不会投入太多资源。我们更愿意抽象出行业共性的能力，把它沉淀到通用模型中，再服务更多客户。</p>
<p>目前，我们发现移动 App 开发者就是一个非常大的市场。现在接近一半的 API Token 消耗都来自这一类客户。</p>
<p label="Q">虎嗅：今年以来，Token 成为了 AI 行业讨论最多的话题之一。很多人认为，AI 视频天然就是一个“Token 黑洞”。你怎么看？</p>
<p label="A">谢旭璋：我觉得，“Token 黑洞”这个说法，前提是它真的创造了价值。如果一个场景只是消耗很多 Token，但没有创造价值，那它就不是一个好的商业模式。真正重要的是，用户愿意持续为这些 Token 买单，因为它创造了更大的价值。</p>
<p>AI 视频就是这样一个场景。现在很多短剧、动漫、影视制作等行业，已经开始大量使用 AI 视频，它们之所以愿意持续消耗 Token，是因为 AI 确实提升了效率，创造了更多商业价值。所以我认为，未来视频一定会成为 AI 行业 Token 消耗最大的几个场景之一，甚至可能就是最大的那个。</p>
<p>很多人会觉得视频 Token 消耗很大，但是其实并不是“贵”的意思，因为它真的有用。</p>
<p label="Q">虎嗅：目前很多 AI 视频产品都采用积分（Credits）计费，而不是像大语言模型一样直接按 Token 计费，这是为什么？</p>
<p label="A">谢旭璋：视频模型和语言模型的底层计算方式其实并不完全一样。</p>
<p>严格来说，视频模型的推理成本并不能简单对应语言模型里的 Token。行业里确实有一些换算方式，但视频生成本身有自己的一套计算逻辑，所以大多数产品最终都会采用视频时长和分辨率 的方式来计费。</p>
<p>我觉得这和是否向用户披露 Token 并没有太大关系。</p>
<p>本质上，它更像是一种成熟的商业模式设计。很多互联网产品都有自己的虚拟货币体系，AI 产品采用积分，其实也是类似的逻辑，让用户更容易理解和消费。</p>
<p>对于 ToB 来说，我们的 API 定价也主要参考行业水平，不会定得最高，也不会刻意价格战，而是保持在行业中间的位置。</p>
<p label="Q">虎嗅：目前 AI 视频产品大多采用订阅制。你认为这会是长期的商业模式吗？</p>
<p label="A">谢旭璋：我觉得订阅只是现在这个阶段最直接、最成熟的一种方式。从长期来看，AI 视频的商业模式一定会越来越丰富。</p>
<p>如果回头看整个视频行业的发展，无论是电影院、长视频平台还是短视频平台，它们最终都形成了不同的商业模式，比如内容付费、广告、增值服务等等。</p>
<p>AI 视频行业其实才刚刚开始。随着模型能力不断提升，用户使用场景越来越丰富，使用时长越来越长，我相信未来一定会出现更多新的商业模式，而不仅仅是今天的订阅。</p>
<p>我们其实已经在一些新兴市场尝试广告加订阅的模式，目前也收到了不错的反馈。所以我觉得，现在整个行业距离真正成熟还有很长的路要走。</p>
<p label="Q">虎嗅：很多人认为，AI 视频 ToC 市场已经很难做了，但你一直强调它的天花板依然很高，为什么？</p>
<p label="A">谢旭璋：我们一直在做一件事情，就是帮助从来没有做过视频的人，完成他们人生中的第一条视频。我们调研发现，平台上一半以上的用户，都是第一次制作视频。</p>
<p>全世界每天都有几十亿人在消费视频，但真正会制作视频的人，占比其实很低，可能只有几个百分点。</p>
<p>过去，大部分人其实都有表达和创作的需求，只是没有能力完成视频制作。AI 正在改变这件事情。</p>
<p>2025 年，我们在全球获得了接近一亿用户，而且平均获客成本非常低。这背后说明，我们满足的是一个过去长期没有被满足的需求。</p>
<p>今天，全世界仍然有几十亿人从来没有制作过视频。如果 AI 能帮助他们更容易地完成第一条视频，并持续进行创作，我认为这个市场还有几十倍的增长空间。</p>
<p>另一方面，我们现在的商业模式其实还相对单一。目前大家主要还是依靠订阅收费，但未来无论是广告、内容分发，还会有更多新的商业模式，都还有很大的想象空间。</p>
<p>所以，无论从用户规模还是商业模式来看，我都认为 AI 视频的 ToC 市场还远远没有到天花板。</p>
<p label="Q">虎嗅：随着业务开始向 ToB 和海外拓展，你们接下来最大的挑战会是什么？</p>
<p label="A">谢旭璋：我觉得最大的挑战其实不是 ToB，而是全球化。</p>
<p>ToB 我们目前主要还是通过标准化 API 去服务客户，它不像传统软件那样需要大量定制开发，也不需要组建很大的销售团队，所以对组织的挑战并没有那么大。</p>
<p>真正新的挑战，是海外本地化。今年我们开始在海外建立团队，也招聘了不少海外员工。未来无论是在东南亚还是欧美，我们都会继续布局，希望能够更好地服务当地用户和客户。</p>
<p>这意味着，我们需要解决跨区域团队协作、沟通以及本地市场运营等问题。不过，我觉得这并不是 AI 行业独有的问题，而是所有希望全球化发展的中国公司都会遇到的挑战。</p>
