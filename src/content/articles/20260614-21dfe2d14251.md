<p><strong> 06月14日 - 06月22日当周重磅财经事件一览，以下均为北京时间：</strong></p>
<p>下周主线事件如下：</p>
<p><strong>首先是，“央行超级周”</strong>：美联储料按兵不动，焦点是新主席沃什的首次发布会——市场关注其将释放鹰派还是鸽派信号，同时是否会“取消点阵图”。同时，日本央行加息概率高达88%，但行长植田和男因病缺席。英国、澳大利亚、印尼、巴西等十国央行亦将密集表态，全球货币政策分化格局进一步演绎。</p>
<p><strong>其次是，中国5月宏观数据</strong>。&nbsp;中国5月社会零售、工业增加值、70城房价等数据集中发布，油价冲击叠加高基数压制，增速或现回落。此外，日本5月CPI、美国5月零售销售、6月纽约联储制造业指数等数据也将出炉。</p>
<p><strong>再次是，科技与地缘的交汇。G7峰会聚焦霍尔木兹局势，OpenAI CEO访韩对接三星</strong>，亚马逊Leo星座执行最大载荷发射，燧原科技、粤芯半导体冲刺科创板。此外，2026陆家嘴论坛将于6月17日至18日召开。</p>
<p>值得注意的是，6月19日周五端午节中国A股、港股、台股休市。美国因6月节也休市。</p>
<h2>经济指标与央行利率决议</h2>
<ul>
<li>
<p><span><strong>美联储利率决议，沃什首秀</strong></span></p>
</li>
</ul>
<p><span><strong>美联储利率决议，沃什首秀</strong></span></p>
<p>北京时间6月18日周四凌晨两点，美联储将公布6月FOMC利率决议，市场普遍预期利率将维持不变。</p>
<p>随后两点半左右，<strong>新任美联储主席沃什将举行其上任后的首次新闻发布会</strong>，市场高度关注这位新掌门人将以何种速度和方式重塑美联储与市场的沟通方式，同时关注其将释放的政策信号。</p>
<p>市场当前讨论的潜在变化包括：<strong>缩短FOMC声明篇幅、取消利率点阵图，以及减少主席新闻发布会频次</strong>。</p>
<ul>
<li>
<p><span><strong>日本央行利率决议，植田和男因病缺席</strong></span></p>
</li>
</ul>
<p><span><strong>日本央行利率决议，植田和男因病缺席</strong></span></p>
<p>日本央行将于6月15至16日举行议息会议。<strong>日本央行行长植田和男因肝囊肿感染住院将缺席此次会议，这是1998年以来首次日本央行行长缺席。市场对本次加息预期已高达88%，行长缺席预计不影响结果。</strong>日本央行同时宣布，副行长内田真一则负责主持16日利率决议会后新闻发布会。</p>
<p>但野村警告，这将令日本央行后续利率路径的政策沟通“极其复杂”。同时，主张宽松政策的首相高市早苗是否会借机影响央行走向，成为市场新的关注焦点。</p>
<p>此外，6月19日周五，日本央行将公布4月货币政策会议纪要。</p>
<ul>
<li><span><strong>英国、澳大利亚、巴西、印尼等央行政策利率</strong> </span></li>
</ul>
<p>下周，<strong>英国央行、澳洲联储</strong>、巴西央行、乌克兰央行、菲律宾央行、印尼央行、瑞士央行、挪威央行、俄罗斯央行、瑞典央行公布利率决议。</p>
<p>市场普遍预计，6月16日澳洲联储将维持利率在4.35%不变。6月18日英国央行预计维持利率在3.75%不变，尽管通胀压力存在，市场广泛预期按兵不动。</p>
<p>新兴市场方面，6月17日巴西央行预计降息25个基点；6月18日印尼央行和菲律宾央行预计各加息50个基点，分别至6%和5%，以应对货币贬值和通胀压力。</p>
<ul>
<li>
<p><span><strong>中国5月社会零售、工业增加值、70城房价等数据</strong></span></p>
</li>
</ul>
<p><span><strong>中国5月社会零售、工业增加值、70城房价等数据</strong></span></p>
<p>6月16日，中国5月社会零售、工业增加值、房地产开发投资、70城房价等数据将出炉。国务院新闻办公室将于当日上午10时举行新闻发布会，介绍5月份国民经济运行情况。</p>
<p>申万宏源证券预计，高油价高通胀对国内消费需求造成冲击，5月社零增速及工业增加值增速或回落。</p>
<ul>
<li>
<p><span><strong>日本5月CPI</strong></span></p>
</li>
</ul>
<p><span><strong>日本5月CPI</strong></span></p>
<p>6月19日，日本将公布5月CPI数据。据彭博经济研究预测，受生鲜食品价格上涨和能源补贴取消推动，整体通胀年率将从4月的1.4%加速至1.6%，核心-核心指标预计维持在1.9%的高位。</p>
